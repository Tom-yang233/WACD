# WACD
Wireless Audio Communication Demonstration Platfrom. For 23Fall USTC DCLT Project-C2.

## Quick Run

### Tx: 
To run **Transmiter**, execute:
```cmd
.\venv\Scripts\activate.bat
python .\src\tx_gui.py
```
or
```cmd
.\tx.bat
```

### Rx:
To run **Receiver**, execute:
```cmd
.\venv\Scripts\activate.bat
python .\src\decode.py
```
or
```
.\rx.bat
```
